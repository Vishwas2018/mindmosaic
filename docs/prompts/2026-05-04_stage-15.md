- Q-15.9: `@mm/engines: workspace:*` peer dep on `@mm/engines-client`.
- Q-15.10: Single commit per BUILD_CONTRACT §11.1.

**Critical structural constraint:**

- `clock` parameter (type `() => number`) must be passed as a function argument to `getTimeRemaining()` and `terminate()`, NOT stored in `EngineState`.
- `EngineState` must be fully JSON-serialisable for the `engine_state_snapshot jsonb` column.
- No `Math.random`, no `Date.now()` inside engine bodies.

**Repo-wide:**

- TypeScript strict; `noPropertyAccessFromIndexSignature` active.
- BUILD_CONTRACT §11.2: no AI Co-Authored-By trailers.
